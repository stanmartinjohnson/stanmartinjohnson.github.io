misc. projects

glossator nugalis
a command-line tool for analyzing and studying ancient greek and latin texts. Built on the classical language toolkit (CLTK), it provides:

normal mode: parse text files or input directly, outputting lemmas, part of speech, and morphological details (CSV-friendly).

flashcard mode: allows you to input large quantities of text directly into the command line whereafter the program will return a two-column, tab-separated list -- the word (column 1) and a compacted parse plus a short english definition (column 2) -- ready to paste into a spreadsheet or save to a file.

this project is first and foremost linux friendly (as of sept. 2025 the project has functionality only in greek)

status: working on it
